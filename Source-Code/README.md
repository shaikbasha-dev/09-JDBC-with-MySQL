# Source Code

Welcome to the **Source-Code** section of the **09-JDBC-with-MySQL** repository.

This directory contains practical Java programs demonstrating how to connect Java applications with MySQL databases using the Java Database Connectivity (JDBC) API.

The examples are arranged in a logical learning sequence, beginning with establishing a database connection and progressing to executing SQL statements, retrieving data, performing CRUD operations, batch processing, and transaction management.

Each source file is designed to explain one specific JDBC concept, making it easier for learners to understand the purpose and implementation of every topic before moving on to the next example.

---

# Objectives

The source code examples in this folder are designed to help learners:

- Understand JDBC fundamentals through practical implementation.
- Learn how Java applications communicate with relational databases.
- Build confidence in writing JDBC programs.
- Practice commonly used database operations.
- Prepare for Java backend developer interviews.
- Strengthen the foundation required for advanced technologies such as Hibernate and Spring Boot.

---

# Learning Sequence

The programs should be studied in the following order.

| Step | Program | Purpose |
|------|---------|---------|
| 1 | ConnectionDemo.java | Establish a connection between Java and MySQL |
| 2 | StatementDemo.java | Execute SQL statements using `Statement` |
| 3 | PreparedStatementDemo.java | Execute parameterized SQL using `PreparedStatement` |
| 4 | ResultSetDemo.java | Retrieve and process query results |
| 5 | InsertRecordDemo.java | Insert records into a database table |
| 6 | UpdateRecordDemo.java | Update existing database records |
| 7 | DeleteRecordDemo.java | Delete records from a database table |
| 8 | SelectRecordDemo.java | Retrieve records using SQL queries |
| 9 | BatchProcessingDemo.java | Execute multiple SQL statements efficiently |
| 10 | TransactionDemo.java | Understand transaction management using commit and rollback |
